🏨 La Posada del Cerro - Documentación del Proyecto
Este sitio web es una solución Premium y Boutique diseñada específicamente para el mercado de hospitalidad de alta gama (Mendoza, Salta, Patagonia, etc.).

🚀 Stack Tecnológico
Frontend: React 18 con Vite (Velocidad de carga ultra-rápida).
Estilos: Vanilla CSS con enfoque moderno (Glassmorphism, Dark Modes).
Animaciones: Framer Motion (Transiciones suaves y premium).
Internacionalización: i18next (Estructura lista para 10+ idiomas).
SEO/Tab Titles: React Helmet Async (Optimizado para Google y UX).
Base de Datos: Firebase (Integración opcional para gestión de habitaciones).
✨ Características Exclusivas
Sistema de Reserva Directa: Formulario inteligente que calcula pax según la habitación y envía todos los datos formateados a WhatsApp (ahorra 15-20% de comisiones de Booking.com).
Seguridad Honeypot: Protección invisible contra bots y spam en formularios.
Blog Integrado: Sección de artículos para posicionamiento orgánico en Google (SEO).
Validación de Fechas: Lógica avanzada para evitar errores de check-in/out.
Diseño Mobile-First: Experiencia perfecta en celulares, que es donde se reserva el 80% de los viajes.
